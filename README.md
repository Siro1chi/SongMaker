# Song Maker

Веб-приложение для создания музыки в браузере.

## Описание

Song Maker — это интерактивный инструмент для написания мелодий и ударных партий. Приложение позволяет выбирать различные инструменты, задавать темп и создавать музыкальные композиции с помощью визуальной сетки.

## Возможности

- 🎵 **Мелодия**: Создание мелодий на различных инструментах
  - 🪕 Лютня
  - 🎻 Виола
  - 📯 Флейта
  - 🔨 Цимбалы
- 🥁 **Ударные**: Программирование ударных партий
- ▶️ **Воспроизведение**: Прослушивание созданной композиции
- 💾 **Экспорт**: Сохранение созданной музыки
- 🎚 **Контроль темпа**: Регулировка скорости воспроизведения (60-200 BPM)
- 📐 **Настройка сетки**: Изменение количества столбцов (1-1024)

## Использование

1. Откройте `index.html` в любом современном браузере
2. Выберите вкладку **Мелодия** или **Ударные**
3. Для мелодии выберите желаемый инструмент
4. Кликните на ячейки сетки, чтобы добавить ноты/удары
5. Настройте темп с помощью ползунка
6. Нажмите **Play** для воспроизведения
7. Используйте **Clear** для очистки сетки
8. Нажмите **Export** для сохранения композиции
